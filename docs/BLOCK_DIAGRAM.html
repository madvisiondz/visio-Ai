§3 — Core product workflow (designation-first)

Never template-first: catalog and images feed pre-selection, then templates, export, and auditable print batches.

flowchart TD
    CSV[Gestium CSV import] --> CAT[(Article catalog Room)]
    PNG[Product PNGs files/product_images] --> IMG[(Image index Room)]

    CAT --> BROWSE[Browse / search Articles designation barcode rayon]
    IMG --> BROWSE

    BROWSE --> PRE[Pre-selection lists]
    PRE --> TMPL[Template / preset Design VisioPRO]
    TMPL --> EXP[Export A4 JPEG PDF PNG]
    EXP --> AUDIT[(Print batches price + designation snapshots)]
