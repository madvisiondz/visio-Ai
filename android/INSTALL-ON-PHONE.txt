INSTALL ON PHONE — Visio Ai
===========================

USE THIS FILE (repo root):

    VisioAI-debug.apk

Rebuild from repo root:

    .\BUILD-APK.ps1

Or from android folder:

    .\dev.ps1 build

Build output (before copy):

    %LOCALAPPDATA%\OasisAI-android-build\outputs\apk\debug\app-debug.apk

App id: com.oasismall.visioai

Uninstall the old app completely before installing a new APK if version/features look wrong.
