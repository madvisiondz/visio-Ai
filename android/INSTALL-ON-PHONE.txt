INSTALL ON PHONE — Oasis AI
===========================

USE THIS FILE (repo root):

    OasisAI-debug.apk

Full path:

    c:\Users\kriim\OneDrive\Desktop\oasis ai\OasisAI-debug.apk

DO NOT use app-debug.apk from android\app\build\outputs\apk\debug\
unless you just built it and copied it to OasisAI-debug.apk.

Why:
- app-debug.apk in build\outputs is often an OLD template ("Hello Android")
  if you have not run a fresh build.
- OasisAI-debug.apk is the copy we use for your phone after assembleDebug.

Rebuild (Android Studio JBR + SDK):

    cd android
    .\dev.ps1 build

That updates OasisAI-debug.apk at the repo root.

Or from Android Studio: Build > Build APK(s), then copy
app\build\outputs\apk\debug\app-debug.apk to OasisAI-debug.apk.

App id: com.oasismall.oasisai

Version 1.1.2+ must show To share / Articles — NOT "Hello Android".
Uninstall old Oasis AI completely before installing the new APK.
